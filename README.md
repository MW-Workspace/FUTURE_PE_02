## 🎯 Campaign Strategy Overview
This repository contains a performance-marketing ad suite designed for **Katrina Cafe & Artisanal Grocery**. Instead of generating generic, over-polished corporate ads that audiences scroll past, this system engineers **User-Generated Content (UGC)** scripts. 

The strategy utilizes native smartphone aesthetics, everyday creator pain points (lack of focus, bad coffee, slow internet), and psychological ad hooks to drive immediate conversion behavior on short-form video platforms like Instagram Reels, TikTok, and YouTube Shorts.

## 🧠 Engineered Prompt Logic
The underlying framework forces the AI engine to function as a professional conversion copywriter by enforcing:
1. **The Multi-Hook Matrix:** Generating three separate structural entry-points (Sensory-driven, Pain-point-driven, Audience-targeted) to enable precise creative A/B testing on ad dashboards.
2. **Dynamic Script Sequencing:** Dividing output into a highly clear, 4-column production layout matching timestamped visual instructions with natural, conversational voiceover audio.
3. **Behavioral Copy Constraints:** Strict negative constraints that ban corporate sales vocabulary, forcing a human-to-human recommendation narrative.
